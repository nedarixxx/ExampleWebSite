# ⚙️ AutoGear Master — Автономный Микро-сайт под ключ https://autogearexample.netlify.app/

Сверхбыстрый, адаптивный статичный мини-сайт (лендинг) для специализированного автосервиса по ремонту АКПП и капитальному ремонту двигателей в Екатеринбурге. 

[cite_start]Разработан в концепции **динамического минимализма** на чистом коде без использования тяжелых CMS (WordPress, Битрикс) и платных конструкторов (Tilda)[cite: 47, 48, 112].

---

## ⚡️ Главные преимущества проекта (Киллер-фичи)

* **0 ₽ за хостинг навсегда:** Сайт разворачивается на глобальной облачной CDN-сети Netlify. [cite_start]Лимиты бесплатного трафика (до 100 ГБ в месяц) полностью закрывают нужды локального малого бизнеса[cite: 14, 61, 63, 112].
* [cite_start]**Запредельная скорость (SEO-эффект):** Время загрузки страниц составляет всего 0.3–0.5 секунды[cite: 79, 116]. [cite_start]Поисковые роботы Яндекса и Google поднимают такой фронтенд выше конкурентов[cite: 79, 116].
* **100% Безопасность и автономность:** В коде нет баз данных и админ-панелей. [cite_start]Сайт физически невозможно взломать, «уронить» или заразить вирусом[cite: 83, 84, 117].
* [cite_start]**Паттерн PRG (Post/Redirect/Get):** Обработка формы защищена от случайных дублей[cite: 144]. [cite_start]При перезагрузке страницы отправки лида браузер не шлет повторный запрос и не спамит владельца бизнеса[cite: 154].
* **Идеальный Mobile-First:** Адаптивная шапка сайта автоматически адаптируется под экраны любых смартфонов (длинный номер телефона на мобильных схлопывается в аккуратную интерактивную иконку-трубку `📞`), полностью исключая горизонтальный скролл.

## 🛠️ Технологический стек

* **Frontend:** Чистый семантический HTML5, современный CSS3 (переменные, Flexbox, Grid Layout, Backdrop-filter blur для эффекта стекла в навигации).
* **Бэкенд форм:** Нативные облачные формы Netlify Forms с поддержкой асинхронной загрузки файлов (фотографий дефектовки деталей) до 10 МБ.  
* **Интеграция карт:** Конструктор Яндекс Карт через изолированный iframe (работает без API-ключей, не тормозит поток загрузки основного сайта и не имеет лимитов на просмотры).

## 🚀 Пошаговый гайд по деплою и запуску

### 1. Загрузка файлов
Загрузите файлы index.html, success.html и изображения в репозиторий GitHub или перетащите папку напрямую в интерфейс Netlify Deploys.  

### 2. Настройка формы auto_repair_order
Форма работает автоматически благодаря тегу data-netlify="true" в коде. Чтобы настроить получение заявок на почту владельца автосервиса:  
1. В панели управления Netlify перейдите в Site configuration -> Notifications -> Emails and webhooks.
2. В блоке Form submission notifications нажмите Add notification -> Email notification.
3. Установите Event на New form submission, укажите email заказчика и выберите форму auto_repair_order.
4. (Опционально) Для отправки лидов в Telegram подключите webhook через коннекторы (Make / Zapier).

### 3. Привязка домена и SSL
1. Вкладка Domain management -> Add custom domain -> Добавьте купленный клиентом домен .ru.  
2. Скопируйте 4 DNS-сервера, которые выдаст Netlify, и вставьте их в личном кабинете регистратора доменов (Reg.ru / Nic.ru).  
3. Подождите 15–30 минут. Хостинг автоматически выпустит вечный бесплатный сертификат безопасности SSL (HTTPS).

## 📝 Как обновлять контент (Инструкция для поддержки)

Сайт легко масштабируется и обновляется. Мелкие правки (изменение цен, замена номеров телефонов или фотографий в галерее) занимают не более 5 минут:  
1. Откройте файл index.html через встроенный редактор GitHub или любой удобный редактор кода (VS Code).  
2. **Изменение цен:** Найдите блок `` и отредактируйте текстовое поле внутри класса .service-price.
3. **Замена фоновых изображений:** Загрузите новые картинки под теми же именами, которые прописаны в коде (Screenshot_6.png и т.д.), либо замените пути к файлам в блоках .hero и .photo-item.
4. Сохраните изменения (Commit). Сайт на живом хостинге обновится автоматически в течение 10 секунд благодаря настроенному авто-деплою.
